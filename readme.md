# Inventory Management API

A simple inventory management system built to learn FastAPI and SQLAlchemy

## Features

- CRUD operations for products
- FastAPI backend
- SQLAlchemy ORM
- PostgreSQL support
- React frontend

## Tech Stack

- FastAPI
- SQLAlchemy
- Pydantic
- React

## Run Locally

```bash
pip install -r requirements.txt
uvicorn main:app --reload
